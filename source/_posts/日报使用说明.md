日报v1.0分为6个打卡目标，exercise，algorithm_p， algorithm_l， review，coding，knowledge

使用自定义scaffold

如果完成了任何一项，则在命令中添加 --checkname true

例如：今日只完成了exercise

```bash
hexo new report test --exercise true
hexo clean
hexo g
hexo s
hexo d

#如果是正常post文章
hexo new post <title>
```
